Документация по программе cube110
===================================

Программа является треннингом.
Способствует запоминанию образно-цифровой системы состоящей из ста
десяти образов.

Цель: запомнить последовательность из 110 образов и их порядковые номера

использованы:

язык программирования:
    **python 3**
GUI:
    **PyQt4**
